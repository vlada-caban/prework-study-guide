HTML

  • HTML stands for Hyperext Markup Language and is used to give content a structure.
  • The <head> element contains the metadata for the web application.
  • Elements that are nested inside other elements are called child elements. The element that holds all the children is known as a parent element.
  • The <body> element represents the visible content shown to the user.

CSS

  • CSS stands for Cascading Style Sheets and is used to give content a style.
  • There are three ways to use CSS: inline CSS, internal CSS style sheet and externas CSS style sheet.
  • Separatiom of concepts is a widely used term in web development and refers to separating code into a different files based on use.
  • CSS box model: Read more here
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.

Git

  • Git is the most commonly used version control software among developers.
  • GItHub is the web interface for Git.
  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it

JavaScript
